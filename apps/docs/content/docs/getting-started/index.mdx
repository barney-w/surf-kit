# Getting Started with surf-kit

surf-kit is an AI-first design system purpose-built for agent interfaces. It provides a complete set of accessible, themeable components for building trust-forward AI experiences.

## What you will find here

- **[Installation](/docs/getting-started/installation)** — Add surf-kit to your project with your preferred package manager.
- **[Quick Start](/docs/getting-started/quick-start)** — Build your first agent chat interface in minutes.

## Prerequisites

- **Node.js** 18 or later
- **React** 18 or later
- **Tailwind CSS** 3.4 or later (for utility-based styling with design tokens)

## Packages

| Package | Description |
| --- | --- |
| `@surf-kit/core` | Core UI components (Button, TextInput, Dialog, etc.) |
| `@surf-kit/tokens` | Design tokens (colours, spacing, typography, radii) |
| `@surf-kit/theme` | Theme provider and dark/light mode support |
| `@surf-kit/hooks` | Shared React hooks |
| `@surf-kit/agent` | High-level agent interface components (coming soon) |

Ready to start? Head to the [Installation](/docs/getting-started/installation) guide.

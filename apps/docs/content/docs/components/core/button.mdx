# Button

A clickable button component with multiple intents and sizes. Built on [React Aria](https://react-spectrum.adobe.com/react-aria/) for full keyboard, screen reader, and focus management support.

## Import

```tsx
import { Button } from '@surf-kit/core';
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `intent` | `'primary' \| 'secondary' \| 'ghost' \| 'danger'` | `'primary'` | Visual style of the button. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the button. |
| `isDisabled` | `boolean` | `false` | Whether the button is disabled. |
| `onPress` | `() => void` | — | Handler called when the button is pressed. |
| `type` | `'button' \| 'submit' \| 'reset'` | — | HTML button type attribute. |
| `aria-label` | `string` | — | Accessible label for icon-only buttons. |
| `className` | `string` | — | Additional CSS classes. |
| `children` | `ReactNode` | — | Button content. |

## Usage

### Basic

```tsx
<Button>Click me</Button>
```

### With intent

```tsx
<Button intent="primary">Primary</Button>
<Button intent="secondary">Secondary</Button>
<Button intent="ghost">Ghost</Button>
<Button intent="danger">Danger</Button>
```

### With size

```tsx
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
```

### Disabled

```tsx
<Button isDisabled>Disabled</Button>
```

### With onPress handler

```tsx
function Example() {
  return (
    <Button onPress={() => console.log('pressed')}>
      Press me
    </Button>
  );
}
```

## Variants showcase

### Primary (default)

The primary button is used for the main action on a page. It draws the most attention with a solid accent background.

```tsx
<Button intent="primary" size="md">
  Save changes
</Button>
```

### Secondary

Secondary buttons are used for supporting actions. They have a transparent background with a visible border.

```tsx
<Button intent="secondary" size="md">
  Cancel
</Button>
```

### Ghost

Ghost buttons are used for tertiary actions or navigation. They have no background or border until hovered.

```tsx
<Button intent="ghost" size="md">
  Learn more
</Button>
```

### Danger

Danger buttons are used for destructive actions like deleting data. They use the error status colour.

```tsx
<Button intent="danger" size="md">
  Delete account
</Button>
```

## Accessibility

- Built on **React Aria** `useButton` for standards-compliant keyboard and screen reader support.
- Supports `Enter` and `Space` key activation.
- `isDisabled` sets `aria-disabled` and prevents interaction.
- Use `aria-label` for icon-only buttons to provide an accessible name.
- Focus ring is visible when navigating with the keyboard (`focus-visible`).
- Disabled state reduces opacity to `0.5` and shows a `not-allowed` cursor.
- Colour contrast meets **WCAG 2.2 AA** standards across all intents and themes.
